# IPL Analytics Dashboard (2010–2024)

This project is an **interactive Power BI dashboard** analyzing IPL seasons from 2010 to 2024.  
It provides insights into teams, players, and match performance.

## Features
- Season-wise winner and total runs/wickets
- Orange Cap (top batsman) and Purple Cap (top bowler) holders
- Individual player performance statistics
- Win percentage by toss decision
- Top 5 bowlers, batsmen, and teams
- Player of the Match analysis

## Technologies
- Power BI
- SQL / Data Modeling
- Python (for data preprocessing)

## Dataset
- IPL match and player data (2010–2024)
